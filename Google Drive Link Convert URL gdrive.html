Google雲端硬碟下載連結轉換器 / Google Drive File Download Link Generator

9月 26, 2016 作品 , Programming/JavaScript 14 Comments Edit Copy Download

(直接跳至Google雲端硬碟下載連結轉換器)

由於Google雲端硬碟可以跨平臺即時線上編修，我現在大部分報告跟投影片都直接使用Google雲端硬碟來製作。特別是最近幾次授課跟演講我都直接改用Google Presentation投影片，只要上課電腦有安裝Google Chrome、能夠正常連上網路，我就可以直接開始演講。

雖然我個人使用很方便，但對其他人來說Google雲端硬碟可沒這麼好用。許多同學還需要下載Google雲端硬碟中的文件，用來列印作為上課筆記的草稿，或是下載PPTX格式回來修改。雖然Google雲端硬碟的編輯界面中有下載或列印的功能，我也寫過一篇「如何從Google簡報下載PDF檔案」來介紹如何操作，但對許多人來說還是太過複雜。最好是只給大家一個連結，直接下載就好了。

於是我參考Jane Vestil這篇教學，製作了一個將Google雲端硬碟的共用連結轉換成各種格式直接下載連結的產生器，提供給大家使用。

如何取得共用連結 / How to get share link?

因為我們要產生一個給大家下載的連結，在這之前我們必須要先開啟Google雲端硬碟的共用設定。

首先，在你要共用的檔案上按右鍵，選擇「共用」。

按下右上角的「開啟連結共用設定」。

這時候Google雲端硬碟會為這份檔案建立一份連結，我們可以按下「複製連結」來取得這個連結即可。

共用連結與可以下載的檔案類型 / Share Link and Export File Types

依照Google雲端硬碟檔案種類的不同，可以產生的下載連結也不盡相同。以下我以四種不同的範例檔案來做說明：

Google文件

  • 共用連結格式：https://docs.google.com/document/d/FILE_ID/edit?usp=sharing 
    舉例：https://docs.google.com/document/d/1HRAvOD8zdX7w6uB15Odd7OpWcZXYG3hvy8BG2TPCmmg/edit?usp=sharing
  • 可下載的檔案類型：PDF、Word (docx)、 OpenDocument Text、Rich Text Format、Text純文字檔、HTML網頁、EPUB電子書

Google試算表

  • 共用連結格式：https://docs.google.com/spreadsheets/d/FILE_ID/edit?usp=sharing 
    舉例：https://docs.google.com/spreadsheets/d/1dOFyNqpjL5K7k-26K-C5wfArAruiYoXcyuXYCkTpbbc/edit?usp=sharing
  • 可下載的檔案類型：PDF、Excel (xlsx)、OpenDocument Spreadsheet、CSV 逗點分隔的資料表、TSV 定位鍵分隔的資料表

Google投影片

  • 共用連結格式：https://docs.google.com/presentation/d/FILE_ID/edit?usp=sharing 
    舉例：https://docs.google.com/presentation/d/1lksB_YpaNv3ZRMe5IysLoIVvJwiNxX_T95vkzYWGEiA/edit?usp=sharing
  • 可下載的檔案類型：PDF、Power Point (pptx)、OpenDocument Presentation、Text純文字檔、JPEG圖片檔、PNG圖片檔、SVG向量圖片檔

其他非Google文件格式的原始檔案

  • 共用連結格式：https://drive.google.com/file/d/FILE_ID/edit?usp=sharing 
    舉例：https://drive.google.com/file/d/0B3_qgRmbvvZ1dVREbzJ0Y29JdG8/view?usp=sharing&resourcekey=0-N9cxqSa0pbE9uoqL33xBcQ
  • 可下載的檔案類型：原始檔案 (如果用共用連結開啟的話，會是線上檢視畫面，而非下載連結)
