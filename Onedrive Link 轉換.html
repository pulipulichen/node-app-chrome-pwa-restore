OneDrive雲端硬碟檔案直接下載連結產生器 / OneDrive File Download Link Generator

2月 22, 2019 程式語言/JavaScript , JavaScript , Programming/JavaScript , Work/Widget 4 Comments Edit Copy Download

除了Google Drive雲端硬碟可以產生直接下載的連結之外，另一個知名的雲端硬碟OneDrive也能夠產生直接下載連結喔。我參考OneDrive 2017 Direct File Download URL Maker這篇的作法，把連結轉換的功能調整成比較順手的操作，並且加入TinyURL短網址功能。讓我們來看看怎麽做吧。

  • OneDrive雲端硬碟檔案直接下載連結產生器

操作步驟 / Instruction

OneDrive檔案直接下載連結要先取得OneDrive的內嵌功能(embed)的程式碼，然後再利用下載連結產生器來產生可以使用的連結。因此以下要分兩步驟來執行：

Step 1. 取得內嵌程式碼 / Get embed code

首先先來到OneDrive的線上網頁中。

對要取得下載連結的檔案按右鍵，選擇「內嵌」。

在產生HTML程式碼以內嵌此檔案下，按下「產生」按鈕。

複製下面的內嵌程式碼，程式碼可能會是這樣子：

<iframe src="https://onedrive.live.com/embed?cid=6BA2FBE8DE6717A9&resid=6BA2FBE8DE6717A9%214868&authkey=AFCoXg0lFBkIE-g&em=2" width="476" height="288" frameborder="0" scrolling="no"></iframe>

這樣就可以進入下一步了。

Step 2. 下載連結產生器 / Generating fIle download link

接下來開啟「OneDrive雲端硬碟檔案直接下載連結產生器」，把內嵌程式碼貼在紅框的位置。

下面就會產生三種不同的連結：

  • Download 下載連結

  • Preview 預覽連結

  • Edit 編輯連結：僅有擁有權限的使用者能夠順利開啟線上編輯頁面

每個連結都有四種功能：

  • COPY LINK 複製連結

  • SHORTEN LINK 建立短連結：使用TinyURL來建立短連結。

  • OPEN 開啟連結

  • 連結網址的文字表單

這樣就可以取得自己需要的連結了。

結語 / In closing

最近因為使用Office來編輯文件，所以試著使用OneDrive來做檔案的同步。跟Google Drive相比，OneDrive的介面做的比較好，但有時候在用Word跟OneDrive同步的時候卻不太穩定。如果用Word儲存到OneDrive當掉的話，就得要從OneDrive重新下載檔案來編輯，有點麻煩。

除了一點小問題之外，OneDrive就是個簡單使用的雲端硬碟。有了直接下載連結產生器之後，我們就可以把檔案放在OneDrive上，提供連結給其他人，讓人方便直接下載，也省下了還要開啟一個確認網頁後再下載的步驟。

舉例來說，如過要寄文件給其他人，又覺得常常需要更新文件內容，那我會把文件擺在OneDrive或Google Drive上，轉換成直接下載連結、並且縮好網址後，將網址放在信件中寄給收件者。日後如果我還要更新文件的話，只要直接更新放在OneDrive或Google Drive上的檔案，再通知對方重新下載一次即可。這樣子不僅省下信箱空間，也可以確保每次下載檔案的時候都會是最新版。

然而，這樣的缺點在於，如果連結上的檔案被我移除的話，對方也就再也無法從這個連結下載檔案了。所以如果收件者拿到的是這樣的下載連結，要記得下載到自己的電腦上保存一份副本，以免網址上的檔案被移除了。
